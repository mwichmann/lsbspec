DSV Questions.

What is a DSV?

A Distribution Software Vendor. An DSV is any organization that produces an installation environment (e.g. Linux distribution) that would be used to install ISV packages on.

As an DSV do I have to supply my distribution packages in RPM format to be LSB compliant?

No. DSV's are not obligated to ship their software in the RPM format. They can use any other format so long as the spirit and letter of the LSB packaging format is honored for installing LSB packages. Supporting ISVs' software by honoring the the LSB recommended format will probably make life easier for themselves and their users while making more software available for use.
