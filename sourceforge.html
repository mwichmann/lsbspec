Linux Standard Base - CVS

How to get check out the site anonymously

  cvs -d:pserver:anonymous@cvs.lsb.sourceforge.net:/cvsroot/lsb login
  cvs -z3 -d:pserver:anonymous@cvs.lsb.sourceforge.net:/cvsroot/lsb co website

Read the file spec/README.SQL for more information on how the specification is built.

Suggested changes are welcome; send diffs to lsb-spec@lists.linuxbase.org.

How to get started as a developer

Well, the way to get started is to check the tree out anonymously (as noted above), and send your diffs to the mailing list. But if you have already done that for a while and someone thinks you should have an account, the procedure is:

  1. register an account at sourceforge.net.
  2. request developer access from Stuart or Dan
  3. check out the tree 
      export CVS_RSH=ssh
  cvs -z3 -ddevelopername@cvs.lsb.sourceforge.net:/cvsroot/lsb co website

Further notes on CVS

After changes to the website are checked in to CVS, they will automatically propagate to www.linuxbase.org within an hour or a few.
