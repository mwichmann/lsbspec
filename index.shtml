The LSB Written Specification

Version 0.1 of the Linux Standard Base Specification can be found here.

Library Reference
This is the inital library reference from Alan Cox. An index of functions in the GNU C library version 2.0 has been extracted to provide the basis of the LSB written reference for C library functions. Most of this will be references to existing standards rather than new work.
