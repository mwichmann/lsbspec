The LSB Written Specification

The Current Draft

The current version of the LSB specification is available both online, and as a RTF file. Additional formats will become available in the future. The current version was last modified

About the Specification

The specification is written in Docbook SGML. Portions of the document are produced from a database. These parts of the document are checked into CVS so that you get a full set of SGML source if you checkout a copy of the document.

There is a top level makefile that contains the commands for building the specifcation from SGML. In short, it runs the one command that is needed to produce the output format that you want. There are several other steps that we run once in a while, so you'll see all of that in the makefile as well.
