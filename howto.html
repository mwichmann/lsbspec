Setup the LSB database users

As the "root" super user for your system, create the MySQL users "lsbuser" and "lsbadmin".   The "lsbadmin" user has the default password of "lf0d" [live free 0r die]

Build the LSB Written Specification

First, you will need to install the Docbook SGML tools for Red Hat 6.2 by running the provided installsgml.sh script. Otherwise, get a copy of the DocBook tools from http://sourceware.cygnus.com/docbook-tools/ .  Install the following packages: docbook, psgml, jade, sgml-common, jadetex, and stylesheets (of course check the documentation at the web site for up to date details).  Also install the sgmltools-lite from Sourceforge.Net located at http://download.sourceforge.net/sgmltools-lite/

Let's build just the SGML portion of the written specification from the data tables that were extracted from CVS.   You will not need MySQL database or the DBI module for Perl to do this

cd lsbspec
make clean
make
Let's build the written specification from the "lsb" database running on your system.  You will need to have the "lsb" database available, plus the DBI module for Perl to do this.
make spotless
make
