Specification: LSB Core generic
Specification: LSB Core IA32, IA64, PPC32, PPC64, S390, S390X, AMD64

Edition of Specification: 4.0
Type: Editorial
Status: Unapproved
Bug reference: http://bugs.linuxbase.org/show_bug.cgi?id=3202

Rationale: the referenced specifications which provide the basis for
the ELF file format specification had hyperlinks which are no longer
valid; this change updates only the hyperlinks, not the referenced
specifications
